Building a minimal research website on GitHub Pages

2025-11-07 · setup, tooling

I wanted a simple, clean site to present my academic work — no frameworks, no JS bundles, no clutter. This post walks through how I built it with pure HTML/CSS and deployed it via GitHub Pages.

Motivation

I wanted to control every pixel and avoid the complexity of static site generators. The goal was a website that loads instantly, is version-controlled, and can be edited offline.

Setup

  • Create a repo named username.github.io.
  • Push your HTML/CSS files to main branch.
  • Enable GitHub Pages in settings.
